Project Description

This Hotel Room Management System is a comprehensive full-stack web application developed as a mini project for the Database Management Systems course. The system provides end-to-end management capabilities for hotel operations including:

  • Customer information management
  • Room availability tracking
  • Booking and reservation handling
  • Payment processing and tracking
  • Staff information management
  • Real-time dashboard with statistics

The application features a modern, responsive user interface with intuitive navigation, real-time data visualization, and secure data handling using SQLite database technology.
